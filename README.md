# 欧洲社科申请雷达

这是一个个人申请季用的静态网站：打开 `index.html` 就能看项目池、截止倒计时、匹配度、材料缺口和个人备注。

## 使用方式

1. 推荐先启动本地服务：
   ```powershell
   py -3 scripts\local_server.py
   ```
2. 用浏览器打开 `http://127.0.0.1:8765/`。
2. 在「个人画像」里改 GPA、IELTS、预算、目标方向和国家偏好。
3. 每个项目卡可以改阶段、加入重点、写备注；这些会存在浏览器本地 `localStorage`。
4. 用「导出」备份你的画像、备注和自定义项目；换浏览器或电脑时再「导入」。

如果你只是直接双击 `index.html`，页面仍然能浏览和筛选项目，但「运行巡检」按钮不能执行本地脚本。

页面里的「申请日历」会按月份整理未来截止日；「材料清单」会按优先级列出 PS、CV、成绩单、语言、推荐信、writing sample 等待办项，勾选状态同样保存在浏览器本地。

## 官网更新巡检

页面里的「官网核验监控」依赖本地脚本更新。进入本目录后运行：

```powershell
py -3 scripts\refresh_sources.py
```

如果页面是通过 `http://127.0.0.1:8765/` 打开的，也可以直接按页面里的「运行巡检」按钮。

脚本会访问 `data.js` 里每个项目的 `source` 链接，生成或更新：

- `sources-status.js`: 页面会读取的巡检状态
- `.source_snapshots/`: 每个官网页面的文本哈希和短摘录

它不会自动改截止日期。学校官网页面结构差异太大，自动抽取 deadline 很容易误判；这个脚本负责告诉你“哪个官网页疑似变动/打不开”，你再打开官网核验日期、学费和材料要求。

只想检查数据文件能否解析、不访问网络时运行：

```powershell
py -3 scripts\refresh_sources.py --dry-run
```

## 数据说明

`data.js` 里内置的是社科申请池模板，不应当直接当作最终申请依据。项目开放日期、截止日期、学费、语言小分、奖学金轮次每年都会变化；卡片上的「待官网核验」就是提醒你把官网信息更新进去。

想改内置项目时，编辑 `data.js` 中对应条目的：

- `rounds`: 开放日期和截止日期
- `requirements`: GPA、IELTS、预算、写作样本等
- `lastVerified`: 你核验官网的日期
- `source`: 官方项目页链接

新增项目可以直接在页面底部添加，也可以复制 `data.js` 里的对象手动维护。

## 从选校 Excel 重新导入

如果你后续继续维护原来的 Excel，可以用导入脚本刷新网站项目池：

```powershell
py -3 scripts\import_selection_workbook.py "D:\欧洲SO联申泛欧选校表-张苏亚萱27fall-04.28更新.xlsx"
py -3 scripts\refresh_sources.py
```

导入会重写 `data.js` 里的项目池，并重置官网巡检状态；第二条命令会重新检查官网链接。
